Welcome to yacs's documentation!
================================
``yacs`` is a tool for simulating centralized job scheduling policies in a distributed
setting.

``yacs`` stands for Yet Another Centralized Scheduler.

It provides 3 scheduling policies: Round Robin (``RR``), Least Loaded (``LL``) and Random (``R``)
