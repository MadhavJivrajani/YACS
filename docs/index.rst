Welcome to the documentation for YACS!
======================================
``yacs`` is a tool for simulating centralized job scheduling policies in a distributed
setting.

It stands for Yet Another Centralized Scheduler and provides 3 scheduling policies: Round Robin (``RR``), Least Loaded (``LL``) and Random (``R``)
